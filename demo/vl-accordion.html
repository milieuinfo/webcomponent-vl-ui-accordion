Accordion via JavaScript openen en sluiten

                
                    <vl-accordion id="accordion-javascript-toggle" toggle-text="Lees meer over de onderwijsdoelstelling">
                        <span>Onderwijs helpt jonge mensen en volwassenen om zichzelf te ontwikkelen en hun weg te vinden in onze samenleving. Het hoger onderwijs speelt daarnaast een belangrijke rol in innovatie dankzij het belang van wetenschappelijk onderzoek.</span>
                    </vl-accordion>

                    <button is="vl-button" onclick="openAccordion(event)">Open</button>
                    <button is="vl-button" onclick="closeAccordion(event)">Close</button>
                    <button is="vl-button" onclick="toggleAccordion(event)">Toggle</button>

                    <script>
                        function getAccordionElement() {
                            return document.querySelector("#accordion-javascript-toggle");
                        }

                        function openAccordion() {
                            getAccordionElement().open();
                        }

                        function closeAccordion() {
                            getAccordionElement().close();
                        }

                        function toggleAccordion() {
                            getAccordionElement().toggle();
                        }
                    </script>
